# Project Title

We have tried to fabricate a WhatsApp-Clone with some of it's
senior features.

![Logo](https://upload.wikimedia.org/wikipedia/commons/6/6b/WhatsApp.svg)

## Installation

Install the WhatsApp-Clone with npm

```bash
  npm install
  npm start

```
